# GATE Exam Dataset Analysis & Classification

## Project Overview
This project analyzes historical GATE exam data and builds a machine learning classification model to predict candidate success using academic and exam-related features.

---

## Objective
- Analyze historical exam data
- Perform feature-based classification
- Build predictive ML models
- Evaluate model performance

---

## Tools Used
- Python
- Keras
- Pandas
- NumPy
- GitHub

---

## Key Tasks
- Data preprocessing and cleaning
- Feature engineering
- Model training using neural networks (Keras)
- Accuracy evaluation and validation

---

## Model Outcome
- Achieved ~90% classification accuracy in predicting candidate success
